How to add a filter

Adding a filter is one of the most effective things that you can do for a photo to make it look good. To add onein pixlr, import your image and select the filter tab at the top. A drop down menu will appear.




Select from the menu which filter you would like and a box on screen will appear, in the box will allow to change the ammount of thr filter. Once the ammount is set to an ammountyou like select Ok and the filter will appear on the image.
