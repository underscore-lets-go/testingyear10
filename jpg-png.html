How to change and image format

To change a photo in JPG format to one of PNG format in Pixlr simply, import your image into the softare and click the file icon in the top left hand corner of the screen. On the drop down menu select save.




In the save box select the format drop down menu, select the format PNG and now select OK. Select your save location and your done!
